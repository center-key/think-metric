Use Template to Create New Article
==================================

1. Open the project in your code editor.

2. Copy the folder:
      src/website/article-list/article-template/
   into:
      src/website/article/

3. Rename the new folder to the article title using
      kebab case (lowercase and dashes).

4. Open the "index.html" file and set the value of the
      "articleTitle" variable to the title of your new
      article using title case.  Save the file.

5. Execute the terminal command:
      $ npm run interactive

6. Visit:
      http://localhost:3000/docs/article-list
   and click on your new article.

7. Edit the files in the folder to complete your article.
